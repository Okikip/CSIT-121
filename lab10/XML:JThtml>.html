  • What is XML?

    Extensible Markup Language (XML) is a language that helps you store and transport data. It is also used to explain what the data is in your own words because XML does not have a definite tag.

  • HOW IS XML USED?

    • To store and describe information
    • XML is used to store the same data in different formats
    • XML is used to specify the structural information of any technical document
    • XML gives structure to data that you see on a web page
