WHAT IS A FORM?

A form is an HTML element that contains and organizes objects called form controls.

Type of form elements

  1. Input Element
  2. Textarea Element
  3. Select Element
  4. Option element
  5. Button element

Input element

  • The input element is a standalone tag. It does not have an opening or closing tag. It is used to input text into a textbox.
